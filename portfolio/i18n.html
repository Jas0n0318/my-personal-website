React 多國語言架構實作

於企業實習期間開發，解決跨國市場的在地化需求

React i18n LocalizationManager JSON
實際操作畫面：支援日文與中文的即時無縫切換

專案背景與挑戰

此專案為協助公司產品拓展至日本市場。由於產品涉及大量電力與能源相關的技術專有名詞，若使用市面上的即時自動翻譯（如 Google Translate 插件），容易造成語意偏差或專業術語錯誤，影響品牌專業度。

因此，我的任務是建立一套「精準控制」的前端翻譯架構，確保每一個顯示在介面上的文字，都是經過人工校對後的正確術語。

技術實作流程

為了達成高效維護，我設計了以下資料流架構，將「程式邏輯」與「翻譯內容」完全分離：

1. UI 文字鍵值化
(Tokenization)
利用 AI 輔助工具，將網頁中原本寫死的文字全部替換成唯一的變數代碼。
2. 建立翻譯字典
(JSON Resources)
建立 JSON 檔案庫，將 Key 值與各國語言（日文/中文）進行對照。這讓非工程師也能輕鬆維護翻譯內容。
3. 狀態管理與渲染
(Localization Manager)
透過 React 的 Context API 建構管理器，監聽語言切換事件，即時從 JSON 撈取對應文字並渲染至畫面。
